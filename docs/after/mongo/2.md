# mvn 安装

安装好jdk,在.bash_profile中配置好java_home,path变量：

```
export PATH=$JAVA_HOME/bin:$PATH
export JAVA_HOME=/Library/Java/JavaVirtualMachines/jdk1.8.0_191.jdk/Contents/Home
export PATH=/opt/apache-maven-3.5.4/bin:$PATH
```
